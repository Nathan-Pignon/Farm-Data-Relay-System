Create and initialize the Display using Wire library

Beware for retro-\/compatibility default values are provided for all parameters see below. Please note that if you don\textquotesingle{}t wan\textquotesingle{}t SD1306\+Wire to initialize and change frequency speed ot need to ensure -\/1 value are specified for all 3 parameters. This can be usefull to control Two\+Wire with multiple device on the same bus.
